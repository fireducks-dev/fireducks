# FireDucks

FireDucks is a dataframe library for python. It is being developed with speed and pandas compatibility in mind.

It is currently available only for Linux (manylinux) on the x86_64 architecture (we will support it for other platforms based on user requests) and it can be simply installed using pip as follows:

```shell
pip install fireducks
```

Please make sure you are using **Python >=3.8, <=3.12**, otherwise you may encounter error saying: “No matching distribution found for fireducks”.

Web site: <https://fireducks-dev.github.io>
